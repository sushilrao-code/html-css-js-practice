🖋Heading tags

📕Definition of Heading Tags

Heading tags in HTML (<h1> to <h6>) are used to define titles and subtitles on a webpage. <h1> is the largest and most important heading, while <h6> is the smallest. They provide structure to the content and help both users and search engines understand the hierarchy of information.

Heading Code Example

Code:

    <h1>This is H1</h1>
    <h2>This is H2</h2>
    <h3>This is H3</h3>
    <h4>This is H4</h4>
    <h5>This is H5</h5>
    <h6>This is H6</h6>

Output:

This is H1

This is H2

This is H3

This is H4

This is H5
This is H6

🤙Paragraph tag

Paragraph tag are used to add paragraphs to an HTML page.

Code Example

<p>This is p </p>

Output
